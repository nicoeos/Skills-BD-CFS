# Pipeline Tracking Template — Carbonfact for Suppliers BD

## Supplier Pipeline Tracker

Use this format to track outreach status across multiple suppliers. Output as a table the user can paste into Notion, Airtable, or a spreadsheet.

### Fields

| Field | Description |
|---|---|
| **Company** | Supplier name |
| **Contact** | Name + title of primary contact |
| **Tier** | 1 / 2 / 3 |
| **Phase** | 0 / 1 / 2 / 3 |
| **ICP Score** | 0–100 |
| **Intro-Call Prob.** | % |
| **Materials** | Key materials/processes |
| **Angle** | A / B / C |
| **Channel** | Email / LinkedIn / Referral |
| **Outreach Date** | Date of initial outreach |
| **Last Touch** | Date of most recent interaction |
| **Status** | Cold / Contacted / Replied / Demo Booked / Profile Live / Active |
| **Next Action** | Concrete next step |
| **Notes** | Objections raised, feedback, referral potential |

### Status Definitions

| Status | Meaning |
|---|---|
| **Cold** | Not yet contacted |
| **Contacted** | Initial outreach sent, no reply |
| **Replied** | Supplier responded (positive, neutral, or negative) |
| **Demo Booked** | Intro call scheduled |
| **Profile Live** | Supplier profile created on platform |
| **Active** | Receiving brand requests, engaged |
| **Parked** | Low priority, revisit on trigger event |

### Weekly Review Checklist

When the user asks for a pipeline review:
1. List all suppliers by status (group by status)
2. Highlight overdue follow-ups (>7 days since last touch without reply)
3. Flag high-ICP suppliers stuck in early phases
4. Suggest next actions for each active lead
5. Identify warm referral opportunities from engaged suppliers
